How to use VIM

What is VIM? 
It's based on shortcuts, called the Vim language, which can make coding and writing faster and more efficient.
With Vim, you can jump to any specific text position and rapidly make precise edits.

Basic VIM Commands:

  1. ":syntax on"=The rules that define the structure of a language
  2. "i" or "a"=Pressing i (for insert) will directly switch to Insert mode. Pressing a (for append) will switch to Insert mode and move the cursor after the current character.
  3. "esc"=Pressing esc will switch from Insert mode to Normal mode
  4. ":wq"=To save and exit VIM.
  5. ":w"=To save VIM
  6. ":q"= To quit/exit VIM

In VIM, you can Ctrl + C to copy and Ctrl + V to paste

In VIM, when you make a typo mistake, it tells you at the bottom "Not an editor command" in red.
